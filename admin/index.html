Welcome to Your Website Manager

Let's connect your admin panel to your website. This only takes a minute and you'll never need to do it again.

1
Create a GitHub token

Click here to create one. Name it Suds Away CRM, set expiration to No expiration.

2
Select your repository

Under "Repository access", select Only select repositories, then choose mike-powerwash-site.

3
Set permissions

Under "Permissions" → "Repository permissions", set Contents to Read and write. Click Generate token.

4
Paste your token below

Copy the token and paste it here. It will be saved in your browser.

Your token is stored only in this browser and never shared.
